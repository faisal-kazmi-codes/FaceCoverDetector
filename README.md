## How to run

### opencv
```
python opencv_dnn_infer.py  --img-path /path/to/your/img
```
### paddle
on image：
```
python paddle_infer.py  --img-path /path/to/your/img
```
### pytorch
on image：
```
python pytorch_infer.py  --img-path /path/to/your/img
```
